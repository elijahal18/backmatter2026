# Back Matter

A digital magazine built with Next.js

---

## How to Run the Site

1. Install Node.js  https://nodejs.org

2. Download this project and open it

3. Open a terminal in the project folder

4. Install dependencies:
npm install

5. Start the site:
npm run dev

6. Open in your browser:
http://localhost:3000

---

## Editing Articles

- Articles are here:
  /app/articles/

- Each article has its own folder:
  /app/articles/exe/page.tsx

- Edit the text directly in those files

---

## Styling

- Colors are in:
  /app/globals.css

- Layout and navigation are in (might want to move both to app):
  /app/layout.tsx
  /content/nav.tsx
